# pachook

Archlinux pacman hooks administrator with disabled exemples

![](http://xn--pp-oia.com/dl/Hg85)

## Installation
With AUR :

    yaourt -S pachook
    
or :

    git clone https://github.com/papajoker/pachook.git
    cd pachook
    makepkg -si

## Usage

pachook {command} "baseHookName"

	 cat
	 edit
	 create
	 enable or start
	 disable or stop
